README – ADIPA (Next.js + SEO Optimizado)
# ADIPA — Plataforma de Cursos de Psicología  
Proyecto construido con **Next.js (App Router)**, **TypeScript**, **TailwindCSS** y optimizado para SEO.

---

## 🚀 Requisitos previos

Asegúrate de tener instalados en tu equipo:

- **Node.js 18+**
- **npm**, **yarn**, **pnpm** o **bun**
- **Git**

---

## 📦 Instalación

Clonar el repositorio:

```sh
git clone https://github.com/CamiloTaborda/landing-adipa-next
cd landing-adipa-next

Instalar dependencias:

npm install
# o
yarn install
# o
pnpm install
🛠 Ejecutar en modo desarrollo
npm run dev

Abrir en navegador:

http://localhost:3000
🏗 Build para producción
npm run build
npm start
